# ML Models

Machine learning models and related utilities.

## Purpose
Traditional ML models for prediction, classification, and analysis (separate from LLM-based agents).

## What Belongs Here
- Trained ML models
- Model loading utilities
- Feature engineering
- Prediction functions

## What Does NOT Belong Here
- LLM interactions (use agents/ with llm_service)
- Data models/schemas (use models/ root)
- Business logic (use services/)
